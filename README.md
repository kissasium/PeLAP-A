# PeLAP-A: Adaptive Latent Pruning for Latent Diffusion Models

Code for PeLAP-A — a latent diffusion model with a learnable channel-wise importance predictor for adaptive latent pruning.

## Contents

- `ALPD_Kaggle.ipynb` — training and evaluation notebook
- `alpd.zip` — source code

## Quick Start (if you want to run it on kaggle)

1. Upload `alpd.zip` contents as a Kaggle Dataset
2. Run `ALPD_Kaggle.ipynb`
